| Command                             | Action                                        |
| :---------------------------------- | :-------------------------------------------- |
| `mix phx.new NAME_OF_APP --no-ecto` | Creates a new project without database (ecto) |
| `mix phx.server`                    | start the created app on localhost            |
| `iex -S mix phx.server`             | load current phoenix app into iex repl        |
